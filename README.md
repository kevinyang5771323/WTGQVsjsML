# 项目简介

协同过滤算法+Springboot Vue游戏百科信息系统是一款基于JavaWeb的综合性游戏百科信息系统。该系统以Springboot和Vue作为主要技术栈，提供了一系列的管理模块和功能，旨在为用户提供一个便捷、高效的游戏信息查询平台。

## 核心功能
- **系统首页**：展示系统概览和信息。
- **用户管理**：管理用户信息、权限等。
- **视频分类**：分类管理攻略视频。
- **攻略视频**：提供攻略视频的上传和管理。
- **攻略分类**：分类管理攻略文章。
- **攻略文章**：上传和管理攻略文章。
- **举报记录**：处理用户举报。
- **意见反馈**：收集用户反馈。
- **游戏论坛**：提供游戏讨论空间。
- **论坛分类**：分类管理论坛讨论。
- **系统管理**：系统设置和管理。

## 适用场景

该系统适用于游戏爱好者、游戏开发者、游戏内容创作者等需要查询和分享游戏相关信息的人群。

## 技术特点

- **架构**：采用B/S架构和MVC设计模式，支持Web端访问。
- **开发环境**：支持IDEA、JDK1.8、Mysql、Maven、Navcat等环境。
- **技术栈**：基于Java、SpringBoot、Vue等主流技术栈，保证了系统的稳定性和易用性。
